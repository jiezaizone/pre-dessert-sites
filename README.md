# pre-dessert-sites

这是一个你懂我懂大家懂的配置文件。大家群策群力，欢迎 pull requests，推动站点配置更新，人人为我，我为人人。

- sites: 有效站点
- closed_sites: 已关闭站点
- all_sites: 用于项目站点配置文件的自动化订阅更新
  - all_sites.txt: CDN 源，无需科学上网，由于缓存机制，更新滞后于 GitHub 源
  - all_sites_github.txt: GitHub 源，需要科学上网
